# Specification

## Summary
**Goal:** Add a fullscreen hamburger menu overlay to the Navbar with animated transitions and navigation links.

**Planned changes:**
- Add a hamburger icon to the Navbar that is visible and clickable on all screen sizes
- Implement a fullscreen overlay that opens when the hamburger icon is clicked, covering the entire viewport with a dark cinematic/glassmorphism background
- Display large navigation links inside the overlay: Services, Portfolio, Pricing, Testimonials, and Contact — each smooth-scrolling to the respective section and closing the overlay on click
- Add an animated close (X) button to dismiss the overlay
- Apply smooth animated entrance/exit transitions (fade-in + slide or scale) for the overlay open/close
- Keep existing desktop navbar links visible alongside the new hamburger icon trigger

**User-visible outcome:** Users on any screen size can click a hamburger icon in the navbar to open a fullscreen animated overlay menu with all main navigation links, and close it via an X button or by clicking a link.
